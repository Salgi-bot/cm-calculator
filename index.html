사용설명서

국토교통부 고시 제2023-580호 · 건설엔지니어링 대가 등에 관한 기준

1. 프로젝트 정보 입력

· 연면적 × 평당단가 → 총 공사비 자동 계산

· 골조비중(%) 입력 → 골조공사비 자동 계산

· 공사기간 입력 → 난이도(D) 자동 산출

· 난이도(D)는 자동값이지만 직접 수정도 가능합니다

노란색 = 직접 입력 회색 = 자동 계산

2. 옵션 설명

· 단순/보통/복잡 → 공종보정계수 Ka (0.9 / 1.0 / 1.1)

· a = Ka 적용 항목   b = 리모델링(Kb) 적용 항목   c = BIM(Kc) 적용 항목

· 감독권한대행 미해당 시 최종 인월수에서 20% 감산

· 안전관리 항목은 난이도 보정 최솟값 = 1.0 (하한 적용)

3. 수량 미세 조정

· 공사기간 변경 시 자동으로 재산출

· 개별 항목은 직접 수정 가능

· 🔄 자동값 초기화 버튼으로 원래대로 복원

4. 저장 및 출력

· 프로젝트 저장/불러오기: JSON 파일 (모든 입력값 보존)

· CSV 저장: 엑셀에서 열 수 있는 형식

· 인쇄/PDF: 프로젝트 개요 + 산식 + 산출 내역 포함
